# Support Matrix

## PHP
- PHP 8.2+
- CI and static analysis are aligned on PHP 8.2 / 8.3 targets where applicable

## Runtime capabilities
- Standalone form builder
- Factory + FormType support
- Validation and submission pipeline
- Object and array data mapping
- HTML rendering with themes
- CSRF protection
- Case-sensitive captcha
- Plugin runtime integration

## Plugin status
- Field type aliases: stable
- Form type aliases: stable
- Extension propagation: stable
- Registry collision handling: stable

## Public stability target
This document reflects the V4.2.0 stable target.
