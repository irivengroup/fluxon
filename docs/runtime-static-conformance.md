# Conformité statique du runtime

## Objectif
Documenter les conventions de typage statique sur la ligne V4.6.x.

## Règles
- toute map runtime doit être annotée en `array<string, mixed>`
- les listes de stages doivent être annotées en `list<string>`
- les métadonnées exposées par le runtime doivent rester structurées et documentées

## Portée
- `FormRuntimeContext`
- `FormRuntimePipeline`
- `FormRenderManager`
- `FormSchemaManager`
